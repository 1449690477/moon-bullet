### 三、当前实现状态

暴走模式下，僚机两侧各有一个激光发射器（精灵贴图），每0.98秒循环一个5阶段周期：

```
warn(0.18s) → fall(0.10s) → sustain(0.32s) → finalburst(0.18s) → dissipate(0.22s)
```

- **warn**：虚线预警线 + 发射口蓄力光球
- **fall**：光束从发射口向目标生长
- **sustain**：全功率光束持续
- **finalburst**：能量爆发（屏震+闪光+顿帧）
- **dissipate**：消散

光束使用锥形梯形 fill（`taperPath`函数），从底部宽到顶端尖，多层叠加：
- 外层柔光晕（bw×6 → bw×0.8）
- 暖金柔光锥（bw×2.8 → bw×0.3）
- 金色主柱（linearGradient沿长度衰减）
- 白金亮带锥
- 核心白柱（收束到针尖）
- 环绕流动光圈（3-4个椭圆环沿光束滑动）
- 顶部源圣环（发射口radialGradient）
- 命中核心（目标处radialGradient）
- 冲击波环（finalburst扩散）

自动瞄准：每周期开始时找屏幕内（y>20）最近的敌人，计算aimAngle。
伤害判定：fall/sustain/finalburst阶段，射线距离检测，每80ms一tick。

### 四、用户的核心不满（必须解决）

用户原话：
> "效果还是不行。缺少冲击光圈的视觉效果。现在的效果像不停闪烁的闪光灯，不像是一个激光炮。我希望是一个激光炮脉冲炮自动去打击敌人的效果。"

**问题诊断**：

1. **"像不停闪烁的闪光灯"**：
   - 根因：光束每1秒循环一次，fall(0.10s)极短→光束突然出现，dissipate(0.22s)快速消失→warn(0.18s)无光束→下一个周期。这个"出现-消失-空白"循环造成了强烈闪烁感。
   - 两个发射器(li=0和li=1)同步周期，同时闪烁，加剧了频闪感。
   - 光束在sustain阶段宽度脉动 `Math.sin(time*18)*0.06` 频率太高（约3Hz），也在闪烁。

2. **"缺少冲击光圈的视觉效果"**：
   - 命中点的冲击波环(Layer 7)只在finalburst阶段(0.18s)出现，太短太弱
   - 没有明显的"光束击中目标"瞬间的冲击扩散感
   - 环绕流动光圈太小太弱，没有"力量爆发的光圈"感

3. **"不像激光炮/脉冲炮"**：
   - 真正的脉冲炮应该是：发射口持续充能发光 → 突然释放一发光束弹 → 光束弹快速飞向目标 → 命中爆开冲击光环 → 发射口继续充能
   - 当前实现是"光束持续存在0.6秒"，更像持续激光而非脉冲炮
   - 缺少"光束弹"的实体感——应该有一个明亮的能量团从发射口射出，而不是一条静态光柱

### 五、要实现的目标效果

参考用户描述："激光炮脉冲炮自动去打击敌人的效果"

**核心视觉**：
1. **发射口常亮**：发射器精灵尖端始终有充能光球（呼吸脉动），不是只在warn阶段才有
2. **脉冲射击**：每隔约1.2秒，发射口能量聚集到最亮 → 释放一个**明亮的光束弹/能量团**沿瞄准方向高速飞向目标
3. **光束弹**：不是静态光柱，而是一个**移动的能量实体**——前端尖锐，尾部拖光，从发射口飞到目标点
4. **命中冲击光圈**：光束弹命中目标瞬间，在目标处爆开**多层扩散冲击光环**（至少3层环从命中点向外扩散，伴随白闪+粒子飞溅+屏震）
5. **光束弹飞行途中**：沿轨迹留下渐隐的光带拖尾（不是持续光柱，而是"飞过的痕迹"）
6. **两发射器错相射击**：li=0和li=1错开半周期，一个射完另一个射，避免同步闪烁

**节奏感**：
- 充能(0.5s)：发射口光球逐渐变亮变大，有"蓄势"感
- 射击(0.15s)：光束弹从发射口飞到目标，速度极快，有"嗖"的穿透感
- 命中爆发(0.3s)：冲击光圈扩散+粒子+屏震+闪光
- 间歇(0.3s)：发射口回到常亮充能状态
- 总周期约1.25s，两发射器错开0.6秒

**冲击光圈规格**（用户最强调的）：
- 命中瞬间：白色核心闪光（radialGradient，半径快速从0扩到60px）
- 第一层环：金色实心环，从命中点扩散到半径80px，0.2秒
- 第二层环：白金环，扩散到半径120px，0.3秒，更细更亮
- 第三层环：淡金环，扩散到半径180px，0.4秒，最淡
- 放射尖刺：12-16条从命中点向外射出的金线，0.3秒
- 粒子飞溅：20+个金色粒子从命中点向四面飞散
- 屏震：screenShake = 8
- 闪光：flash = 0.3
- 顿帧：triggerHitStop(0.04)

### 七、建议的实现方案

```
阶段重新设计：
  charge(0.50s) — 发射口充能光球逐渐变亮变大（常亮基础光球+增长层）
  fire(0.15s)   — 光束弹从发射口飞向目标（移动的能量实体+拖尾）
  impact(0.30s) — 命中冲击光圈扩散（3层环+尖刺+粒子+屏震+闪光）
  cooldown(0.30s) — 发射口回到常亮状态，准备下一发
  总周期1.25s，li=0和li=1错开0.625秒
```

**光束弹（fire阶段）实现思路**：
- 不是画一整条光柱，而是画一个**沿aim方向移动的发光体**
- 发光体位置：`pos = lerp(emitPos, targetPos, fireProgress)`
- 发光体本身：radialGradient核心白 + 外层金芒 + 前端尖锐三角
- 拖尾：从发射口到当前发光体位置的渐隐光带（alpha从1→0）
- 发光体飞过时在路径上留下渐隐粒子

**冲击光圈（impact阶段）实现思路**：
- 在targetPos处绘制多层扩散环
- 每层环：`ringRadius = startR + (endR - startR) * progress`
- alpha = `(1 - progress) * 强度`
- 用stroke画环 + fill画核心闪光
- 至少3层不同速度/半径/颜色的环

**发射口常亮（charge+cooldown阶段）**：
- 一个呼吸脉动的光球始终在发射口
- `coreR = lw * 0.3 * (1 + Math.sin(time*4) * 0.15)`
- radialGradient：白心→金边→透明

## 🎨 东方STG设计原则应用

### 1. 负空间（Negative Space）
✅ 8方向环 → 8个明确的缝隙
✅ 扇形弹幕 → 扇形两侧是安全区
✅ 螺旋臂 → 螺旋之间有规律空隙

### 2. 可预判性（Predictability）
✅ 固定角度步进（螺旋：0.15-0.18 rad）
✅ 固定旋转速度（环形：0.15 rad/s）
✅ 无随机元素
✅ 固定发射间隔

### 3. 时空密度控制
✅ 空间密度降低70%
✅ 时间密度降低（间隔增加30-44%）
✅ 错开发射时机（根据敌人X位置）

### 4. 引导性设计
✅ 弹幕"画出"安全路径
✅ 对称设计引导选择
✅ 规律节奏可以"跟随"

```javascript
// 简单模式

patFan5(e, spread, speed, color, r, kind)
patRing8(e, speed, color, r, kind, angleOffset)
patRing12(e, speed, color, r, kind, angleOffset)
patSpiral2(e, speed, color, r, kind)
patSpiral3(e, speed, color, r, kind)
```

### 自动角度累积

```javascript
// 螺旋自动管理角度
if (!e.spiralAngle) e.spiralAngle = 0;
// ... 发射 ...
e.spiralAngle += 0.18; // 固定步进
```

### 发射时机错开

```javascript
fireTimer: 1.0 + (x / W) * 0.8
// X位置越靠右，首次发射越晚
// 形成"波次"而非"墙"
```

## 🎮 游戏体验预期

### 新手（Stage 1-2）
- bat单发：最简单，学习瞄准和移动
- familiar扇形：学习左右躲避
- wisp螺旋：学习观察规律
- 开火间隔长：有充足反应时间

### 中级（Stage 3）
- nun环形：学习穿插缝隙
- cherub扇形：更密集的选择
- 仍然规律可预判

### 高级（Stage 4 + Boss）
- seraphguard三臂螺旋：复杂但规律
- warden组合攻击：多重威胁
- Boss多阶段：递进挑战
- 但永远不会"满屏乱飞"
